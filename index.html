Convenciones iniciales

Reglas para seguir aterrizando Academy como libreria.

Un entrypoint

Todos los HTML consumen `academy-catalogo.css` y `academy-catalogo.js`.

Capas CSS

Tokens, base, layout, componentes, patrones, login, catalogo y reglas especificas Academy.

Acciones funcionales

Las vistas son estaticas, pero tienen navegacion, formularios, filtros, toasts y modales demo para validar manejo antes de integrar API.

Guia de migracion

El archivo `MIGRACION-UI.md` cruza cada ruta base con la vista propuesta y sus mejoras.
